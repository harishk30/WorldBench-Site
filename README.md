# WorldBench Website

Static project website for **WorldBench: A Challenging and Visually Diverse Multimodal Reasoning Benchmark**.

## Preview

Open `index.html` directly, or run a local server:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## Deploy

This repo is ready for GitHub Pages. Push it to a repository and enable Pages from the repository root.
